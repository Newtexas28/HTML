Forklaring:

Appen over skal fungere som ein enkel kalkulator å gjere rekne operasjoane plus, minus, gange og dele.

Det eg gjorde var å hente inn input verdiar å deklarere dei til variablar. Så lager eg ein ny variabel med rekne operasjonen, før eg sender verdi tilbake.

Eg lagde også ein button til kvar av rekneoperasjonane som kjøyre den spesefikke rekne operasjonen og sender svaret tilbake til websida
