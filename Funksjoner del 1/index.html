Funksjoner er ein gjenbrukbar del av kode som putter saman ein spesefikk sekvans av for å gjere ein spesefikk oppgåve. Funksjoner inneheld argumenter som kan blir brukt inn i funksjonen. Døme: const klikk = (input1, input2) => { return input1 * input2; };
